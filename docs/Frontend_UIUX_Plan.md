# PharmaAssist Frontend UI/UX Plan

## Document Overview
This document outlines the complete frontend architecture, user experience flows, and UI design specifications for the PharmaAssist pharmaceutical distribution and e-pharmacy platform.

**Target Market:** Bosnia and Herzegovina  
**Technology Stack:** Angular 21, Angular Material, TailwindCSS  
**Languages:** Bosnian (Primary), English (Secondary)

---

## Part 1: User Roles & Access Matrix

### 1.1 System User Roles

| Role | Description | Primary Access Areas |
|------|-------------|---------------------|
| **SuperAdmin** | System administrator with full access | All modules, system settings, user management |
| **Admin** | Company administrator | Most modules except system settings |
| **Manager** | Department/Branch manager | Inventory, orders, reports, staff oversight |
| **Pharmacist** | Licensed pharmacist | Prescriptions, dispensing, patient counseling |
| **Sales Rep** | Field sales representative | Customers, orders, limited inventory view |
| **Warehouse** | Warehouse staff | Inventory, stock movements, receiving |
| **Customer** | E-pharmacy customer (B2C) | Shopping, orders, prescriptions, account |
